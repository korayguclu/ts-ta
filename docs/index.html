ts-ta

TS-TA

一个使用typescript写的纯javascript技术分析库。

Install

npm install --save ts-ta

Features

  • MA 移动平均线
  • ASI 振动升降指标
  • DMA 平均线差
  • DMI 动向指数
  • EXPMA(EMA) 指数平均数
  • MACD 平滑异同移动平均线

更多指标持续更新中。。。
